# Installation

Clone the repository:
```
git clone https://github.com/mh1775089-lgtm/Website-parameter-scanning.git
```
Go to the project directory:
```
cd noobxvau-scanner
```
Install required packages:
```
pip install -r requirements.txt
```
Run the tool:
```
python scanner.py
```
